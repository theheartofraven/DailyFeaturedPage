DailyFeaturedPage
=================

This mediawiki extension lets you show transcluded article content on different days.  For example,  lets say on your main page, you want to showcase content from selected articles and rotate different article text each day, you can use this extension to program a different article text to appear each day  ("the daily featured page").  You can also use modify this extension by using  other PHP date functions  so that the featured page content is shown on an hourly or minutely basis.  More info at http://www.mediawiki.org/wiki/Extension:DailyFeaturedPage
